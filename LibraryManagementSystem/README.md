# Library Management System

This project is a console-based Library Management System written in C#. It was created as part of the final project for the Coursera course **[Introduction to Programming with C#](https://www.coursera.org/learn/introduction-to-programming-with-c-sharp/home/info)**.

## About the Course
The course, offered on Coursera, introduces the fundamentals of programming using C#. It covers topics such as:
- Variables and data types
- Loops and conditionals
- Methods and modular programming
- Object-oriented programming concepts

This project demonstrates the application of these concepts in a real-world scenario.

## Features
- **Add and Remove Books**: Manage the library's collection.
- **Search for Books**: Find books by title.
- **Borrowing Limit**: Users can borrow up to 3 books at a time.
- **Return Books**: Return borrowed books to make them available again.
